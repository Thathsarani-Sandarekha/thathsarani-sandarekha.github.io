Edge AI Parking System

Real-time parking management system with license plate detection (YOLOv8), plate reading (PaddleOCR), FastAPI backend services, and a Flutter UI — deployed on an edge device with low-latency slot updates.

Problem

Manual parking tracking is slow and error-prone. Traditional cloud-only LPR systems also struggle with latency and connectivity. This project solves real-time license plate recognition and slot updates directly on an edge device.

Solution

A complete edge-to-app pipeline: YOLOv8 detects plates, PaddleOCR reads text, and FastAPI serves results to a Flutter UI that updates slot availability in real time.

Results

Plate Recognition Accuracy 95% Successful end-to-end detection + OCR reading
Slot Update Latency < 1s Near real-time UI updates from edge device
Deployment Edge Runs locally without depending on cloud connectivity

What I built

  • YOLOv8 plate detection pipeline + OCR-based plate text extraction
  • FastAPI backend APIs for recognition results + parking slot state management
  • Flutter UI for slot visualization, vehicle entry/exit, and live updates
  • Edge deployment workflow and end-to-end integration testing

Quick Facts

  • OCR: PaddleOCR
