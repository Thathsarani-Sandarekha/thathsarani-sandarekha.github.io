What it is

A file-first AI analytics agent that answers natural-language questions over multi-room NDJSON air-quality logs, returning tables, charts, and plain-English summaries — without a database.

Why it matters

Turns messy sensor logs (inconsistent field names across rooms) into instant insights using LLM-guided code generation + safe execution.

Highlights

  • Semantic normalization for inconsistent columns (e.g., CO2 (ppm) vs CO₂).
  • LLM decides: table / chart / both.
  • Pandas + Matplotlib code generation and controlled execution.
  • Clean error handling and formatted outputs.

My role

Built the end-to-end pipeline (ingestion → normalization → LLM planning → code exec → summary) and integrated backend + UI.

Quick Facts

  • Category: AI Agent / Data Analytics
  • Model: LLaMA3-70B (Groq)
  • Backend: FastAPI
  • Analytics: Pandas, Matplotlib
  • Input: NDJSON (per room)
  • Output: Summary + Table + Chart
