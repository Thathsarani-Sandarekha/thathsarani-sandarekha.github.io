What it is

ChatGroq is a multi-document conversational QA system. Users upload PDFs, CSVs, and Excel sheets together, ask questions in natural language, and the chatbot retrieves the most relevant chunks before answering — showing file-backed sources for trust and transparency.

Highlights

  • Multi-file upload: PDF + CSV + XLS/XLSX in one session.
  • Chunking + embeddings + FAISS similarity search for fast retrieval.
  • LLM answers restricted to retrieved context (reduces hallucinations).
  • Displays retrieved chunks and file sources to validate answers.
  • Built to extend into multimodal RAG (image reasoning via LLaVA).

Stack

Streamlit, LangChain, FAISS, HuggingFace Embeddings, Groq API (LLaMA3).

Quick Facts

  • Category: RAG / Document QA
  • Inputs: PDF, CSV, Excel
  • Embedding: HuggingFaceEmbeddings
  • Vector Store: FAISS
  • LLM: Groq LLaMA3
  • UI: Streamlit

Current upgrade

Extending to a multimodal RAG system with support for image-based reasoning using LLaVA (e.g., screenshots, charts, image-heavy PDFs).
