What it does

  • Runs sentiment classification on Yelp review text.
  • Compares Multilingual BERT vs Twitter RoBERTa (sentiment).
  • Outputs predicted label and confidence scores (probabilities).
  • Saves trained / prepared model artifacts for reuse.

Why two models?

Yelp reviews may contain mixed language usage and informal expressions. mBERT supports multilingual understanding, while RoBERTa (sentiment) is optimized for nuanced English sentiment patterns — the comparison highlights practical tradeoffs.

Pipeline summary

Review Text → Tokenization → Transformer Inference (BERT / RoBERTa) → Sentiment Label + Confidence → Save Models

Deliverables

  • Reusable saved models (Google Drive link).
  • Clean codebase for running inference and comparison.
  • Documentation describing the models and use-case.
