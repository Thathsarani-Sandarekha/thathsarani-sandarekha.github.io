What it does

  • Detects suspicious events from video: weapon presence, violence, and intrusions.
  • Runs in real time with bounding boxes and on-screen indications.
  • Supports alert-triggering logic for surveillance monitoring workflows.

System flow

Video Stream → Frame Extraction (OpenCV) → YOLOv8 Detection → Event Logic (weapon/violence/intrusion) → Visual Overlay + Alerts

Results

Violence Detection F1 = 0.92 Balanced precision/recall for violence events
Weapon Detection Accuracy = 0.85 Correct weapon presence classification
