Brain Tumor Segmentation & Classification

A dual-stage deep learning pipeline that mimics real clinical workflows by first segmenting tumors using U-Net and then classifying tumor types using CNNs from MRI scans.

Problem

Manual analysis of brain MRI scans is time-consuming and prone to variability. Accurate tumor localization and classification are critical for diagnosis and treatment planning.

Solution

This project implements a dual-stage pipeline where a U-Net model first isolates tumor regions, and a CNN classifier then predicts the tumor type from the segmented region—closely reflecting clinical decision workflows.

Results & Evaluation

Segmentation Dice Score 0.89 Overlap accuracy between predicted and ground-truth tumor masks
Classification Accuracy 94.5% 4-class brain tumor classification performance
Macro F1-Score 0.95 Balanced performance across all tumor classes
