MetaFaces

Dynamic Memory Networks for consistent multi-character representation in multi-image visual storytelling. Built on Stable Diffusion XL with anchor-based feature injection and attention-guided segmentation using Grounding DINO + SAM (Grounded SAM).

Problem

Diffusion story generation suffers from character drift across scenes (identity + appearance changes), especially with multiple characters.

Solution

MetaFaces uses anchor-based masked feature storage in a dynamic memory module, then injects retrieved features during denoising using attention guidance. Segmentation is done via Grounded SAM.

Results

CLIP Score 0.3619 Semantic similarity (anchor ↔ scenes)
DreamSim Score 0.1753 Perceptual identity consistency

What I built

  • Memory module for masked character feature tensors
  • Anchor generation + Grounded SAM segmentation flow
  • Attention-guided feature injection during denoising
  • Evaluation pipeline using CLIP + DreamSim
